# 🤖 AI-Powered Todo Agent (Gemini & FastAPI)

פרויקט זה מציג סוכן בינה מלאכותית (AI Agent) חכם לניהול משימות. הסוכן מסוגל להבין פקודות בשפה חופשית, להסיק אילו פעולות עליו לבצע, ולהפעיל פונקציות מערכת אמיתיות כדי לנהל רשימת מטלות.

---

## 🚀 תכונות עיקריות
* **הבנת שפה טבעית:** שימוש במודל `gemini-1.5-flash` לניתוח בקשות משתמש.
* **ניהול משימות מלא (CRUD):** הוספה, שליפה, עדכון ומחיקה של משימות.
* **מבנה נתונים עשיר:** תמיכה בכותרת, תיאור, סוג משימה, תאריכי התחלה וסיום וסטטוס.
* **ממשק React מודרני:** צ'אט מעוצב וחוויית משתמש אלגנטית בקו נקי.
* **Backend חזק:** מבוסס FastAPI עם חיבור אוטומטי לפונקציות (Function Calling).

---

## 🛠 טכנולוגיות
* **Backend:** Python 3.x, FastAPI, Uvicorn.
* **AI Model:** Google Gemini API (google-generativeai).
* **Frontend:** React, Tailwind CSS (via CDN).
* **Security:** Python-dotenv (ניהול מפתחות API).

---

## 📂 מבנה הפרויקט
* `todo_service.py` - לוגיקת המשימות (ניהול הנתונים במערך).
* `agent_service.py` - חיבור ל-Gemini ומימוש ה-Agent.
* `main.py` - שרת ה-API שמרכז את הכל ומטפל ב-CORS.
* `.env` - קובץ הגדרות למפתח ה-API (לא להעלות לגיט!).
* `todo-client/` - תיקיית ממשק המשתמש (React).
